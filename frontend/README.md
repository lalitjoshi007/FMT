## 🚀 Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/flatlogic/react-native-starter.git
   cd react-native-starter
   ```
2. **Install dependencies**
   ```bash
   yarn install
   ```
3. **Run the app**
   ```bash
   yarn run:ios  # for iOS
   yarn run:android  # for Android
   ```
